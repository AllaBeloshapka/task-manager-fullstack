Instructions for use
TaskCounter


A simple web app for adding, deleting, and tracking daily tasks.
It allows you to track, view, and delete tasks throughout the day,
as well as see the current number of active tasks.

To-do list.


Opposite each task there will be a small square.
Check the box next to the task you want to delete.
It will appear in the input field.
Click the "minus" sign - the task will disappear from the list and the counter will decrease.
Enter the task text in the input field.
